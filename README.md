# Technology Products Revenue Dashboard

## Project Overview
This project is an interactive Microsoft Excel dashboard created to analyze revenue performance across different product categories and technology products. The dashboard provides insights into category revenue, product-level revenue, and the relationship between customer ratings and revenue generation.

Using Pivot Tables, Pivot Charts, and Slicers, the dashboard allows users to filter data dynamically and explore business performance from multiple perspectives.

## Objectives
- Analyze revenue by product category.
- Compare revenue across different products.
- Evaluate the impact of product ratings on revenue.
- Provide an interactive reporting solution for business analysis.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Dashboard Features
- Revenue by Category Analysis
- Revenue by Product and Category
- Revenue by Product Ratings
- Interactive Product Filters
- Dynamic Category Selection

## Key Insights
- Electronics generated the highest revenue among all categories.
- Smartphones and laptops were among the strongest revenue contributors.
- Higher-rated products showed strong revenue performance.
- Revenue distribution varies across product categories, helping identify profitable areas.
